# java-jdbc-connectivity-
inventory managemet system

**Overview**
A Java-based Inventory Management System developed using JDBC and MySQL to perform basic inventory operations such as adding, updating, viewing, and deleting products.

**Technologies Used**
Java
JDBC
Oracle 
JDK 8+
Features
Add, update, delete products
View inventory records
Database-driven storage

**How to Run**
Import the project into a Java IDE(NETBEANS)
Configure database credentials in JDBC connection file
Run the SQL script
Execute the main Java file
